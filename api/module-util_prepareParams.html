Module: util/prepareParams

Assembles a parameter list for a prepared statement from a list of records being inserted or updated. Parameter list generation for WHERE clauses in all statements is handled in module:where.
Parameters:
Name Type Description
fields Array A list of fields to pick from each record.
records Array A list of records.
Source:
  • lib/util/prepare-params.js, line 5
Returns:
An array containing the values of each key from each record, or undefined if the record does not contain the key.
