Module: util/decompose

Collapses tabular result sets into a hierarchical object graph based on the provided schema.
Parameters:
Name Type Description
schema Object An object describing the shape of the output data. Schema objects are nested to arbitrary depth, with a single top-level schema describing the "origin" record format and containing other schemas which describe descendant records.
Properties
Name Type Description
pk String The name of a field in the result set which uniquely identifies a single record.
columns Object | Array A map of field names in the result set to keys of an output object, or an array of field names if no transformation of names is desired.
array Boolean When true, entities built with this schema are coerced into arrays of entities. This is only valid for nested schemas. If an object described by this schema's parent schema does not contain any records for this schema, an empty array will be generated.
anythingElse Object Nests a descendant schema under this one.
data Array Array of database records to decompose.
Source:
  • lib/util/decompose.js, line 5
Returns:
An array of nested objects generated by applying the schema to each data element in turn.
